회원 관리 프로그램

-- 기능
1. 회원 등록
2. 회원 목록 보기
3. 종료
4. 회원 정보 수정 기능.(추가)
5. 회원 정보 삭제 기능.(추가)

-- 프로그램 규칙
1. 프로그램을 실행하면 /users.dat 파일에서 회원정보를 읽어온다.
2. 해당 파일이 없을 경우 읽어 오지 않는다.
3. 프로그램 종료시, 메모리에 있는 회원 정보를 /users.dat에 저장한다.
4. 프로그램을 강제 종료하면 입력된 회원 정보는 저장이 안될 수도 있다.

-- 필요기능
1. java I/O에서 외부 파일을 불러오고 작성후 다시 저장할 수 있는 기능.
2. 파일이 없을때 안 불러오게하는 코드
3. 프로그램 강제종료시 입력된 정보가 저장안될 수 도 있다.
